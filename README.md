# xd-cli
## 预想实现功能
- 拿本地IPv4`ip`☑️
- 静态起服务`server`☑️
- 压缩图片资源`rar`(速度较快，压缩效果不及tinify，且暂时不能传参)☑️
- 压缩图片资源`tinify`(速度较慢，压缩效果好，支持网路图片压缩后存本地`tinify http://asdf.png`)☑️
- 快速生成项目模版`xd ls`☑️
- 自动化爬虫
- 图片自动上传七牛云返回cdn链接

## 项目模版是主要功能点
- 移动端h5-vue
